# KAN-2-challenges

## Challenge 1: Temperature Converter

Create a program with functions that convert temperatures between Celsius, Fahrenheit, and Kelvin.

Create separate functions: celsius_to_fahrenheit(), fahrenheit_to_celsius(), celsius_to_kelvin(), kelvin_to_celsius()

Each function should take one temperature value as input and return the converted value

Create a main menu function that lets the user choose which conversion to perform

Test with at least 3 different temperature values

## Challenge 2: Grade Calculator

Write a program that calculates student grades using functions.

Create a function calculate_average() that takes a list of scores and returns the average

Create a function get_letter_grade() that takes a numerical grade (0-100) and returns the letter grade (A, B, C, D, F)

Create a function display_report() that takes a student name, list of scores, and displays their average and letter grade

Test with at least 2 different students

## Challenge 3: Number Analysis Tool

Build a program that analyzes numbers using multiple functions.

Create a function is_prime() that checks if a number is prime and returns True/False

Create a function get_factors() that returns a list of all factors of a number

Create a function sum_of_digits() that returns the sum of all digits in a number

Create a main function that prompts the user to enter a number and displays all three analyses

Test with numbers like: 17, 28, and 100
